# Exercícios de Lógica de Programação

Este repositório reúne diversos exercícios desenvolvidos durante meus estudos de lógica de programação, utilizando Portugol no Visualg.

---

# Objetivo

O objetivo deste repositório é fortalecer a base em programação por meio da prática de algoritmos e resolução de problemas.

---

# Conteúdos abordados

- Estruturas condicionais (if, else)
- Estruturas de repetição (para, enquanto)
- Vetores
- Matrizes
- Operações matemáticas
- Entrada e saída de dados

---

# Tecnologias utilizadas

- Portugol (Visualg)
- Git
- GitHub
